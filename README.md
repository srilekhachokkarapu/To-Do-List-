# 📝 To-Do List App

A simple and responsive To-Do List web application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add new tasks
- View your task list
- Clean and modern UI design
- Responsive layout
- Beginner-friendly code structure

## 🌐 Technologies Used
- HTML5
- CSS3
- JavaScript
